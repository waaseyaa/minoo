# WP02 — EntityContentProvider cluster (OralHistory, Teaching, Event)

## Objective

Migrate the 3 content-bundle entities owned by `EntityContentProvider`
to explicit `tenancy: ['scope' => 'community']`. Same pattern as WP01,
applied to a different provider.

## Context

WP01 has already merged to `main` and validated the pattern. This WP
follows the exact same shape — only the file paths and entity names
change. Before starting, read WP01's PR diff to confirm the implemented
pattern; if WP01 deviated from the planning artifacts, replicate the
deviation here.

Read [spec.md](../spec.md), [plan.md](../plan.md), and
[data-model.md](../data-model.md). The same constraints apply: C-001
(no behavioral change), C-002 (atomic per-entity edit + provider edit),
C-003 (alpha.173+ floor).
