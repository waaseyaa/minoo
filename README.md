# Minoo

Indigenous knowledge platform powered by Waaseyaa CMS and NorthCloud ingestion.

## Development

```bash
composer install
php -S localhost:8081 -t public
```

## Testing

```bash
./vendor/bin/phpunit --testsuite MinooUnit
./vendor/bin/phpunit --testsuite MinooIntegration
```
